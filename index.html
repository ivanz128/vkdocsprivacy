VK Docs application privacy policy. Политика конфиденциальности приложения ВК Документы.

VK Docs application does stores user's ID in VK social network in the device memory for the purpose of showing correct user data (list of documents). Also the application stores list of documents from user's VK profile for the purpose of offline access (caching). User's password is not stored in VK Docs since it uses OAuth2 authorization via official VK app.
No data is being stored on server, VK Docs only stores user data in the device memory.

Приложение ВК Документы хранит ID пользователя в социальной сети Вконтакте с целью отображения корректной информации пользователя (списка документов). Также приложение хранит список документов из профиля пользователя с целью обеспечения оффлайн доступа (кеширование). Пароль пользователя не хранится в ВК Документы, поскольку для авторизации используется протокол OAuth2 и официальное приложение Вконтакте.
Никакая информация не хранится на сервере, ВК Документы хранит информацию исключительно в памяти устройства.
